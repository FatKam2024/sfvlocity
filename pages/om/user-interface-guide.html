User Interface Guide

Learn to navigate and use the Salesforce Vlocity OM interface efficiently to manage orders and related tasks.

Introduction

The Salesforce Vlocity OM user interface is designed to provide a seamless experience for managing the entire order lifecycle. Familiarity with this interface is crucial for efficient order management.

Key Interface Elements

  • Order Management Console: Central hub for managing orders.
  • Order Detail Page: Comprehensive view of individual orders.
  • Product Catalog: Interface for browsing and selecting products.
  • Order Workflow Visualizer: Graphical representation of order status and progress.

Hands-On Lab: Navigating the OM Interface

This lab will guide you through the key areas of the Salesforce Vlocity OM interface.

Lab Steps

  1. Log into your Salesforce org with Vlocity OM installed.
  2. Navigate to the Order Management Console.
  3. Create a new order and explore the order creation process.
  4. Use the Product Catalog to add items to your order.
  5. Review the Order Detail Page and understand the information displayed.
  6. Explore the Order Workflow Visualizer to understand the order's progress.

Best Practices for Using the OM Interface

  • Customize your view in the Order Management Console to display the most relevant information for your role.
  • Use quick actions and keyboard shortcuts to improve efficiency in order processing.
  • Regularly review and update your saved searches and list views to quickly access relevant orders.
