User Interface Guide for Salesforce Vlocity

Explore the user interface of Salesforce Vlocity, understanding the key components and how to navigate the platform efficiently.

Introduction

The user interface (UI) is the front end of Salesforce Vlocity, where users interact with the system. This guide will help you get acquainted with the layout, components, and best practices for navigating the UI.

Key UI Components

  • Navigation Bar: Access main modules and frequently used tools.
  • Home Page: The dashboard that provides an overview of key metrics and tasks.
  • Record Pages: Detailed views of individual records, including products, quotes, and orders.
  • Global Actions: Quick access to commonly used actions, such as creating a new record.
  • Search Functionality: Efficiently find records, reports, and more within Salesforce Vlocity.

Hands-On Lab: Exploring the User Interface

This lab will guide you through the key features and navigation options within the Salesforce Vlocity user interface.

Lab Steps

  1. Log into your Salesforce org with Vlocity installed.
  2. Familiarize yourself with the navigation bar and home page.
  3. Explore a sample record page and understand its components.
  4. Use the global actions to create a new record.
  5. Search for a specific record using the search functionality.

Best Practices for UI Navigation

  • Customize the navigation bar to include the tools and modules you use most frequently.
  • Regularly review the home page to stay updated on key metrics and tasks.
  • Use the search function to quickly find records and reports, saving time.
  • Leverage global actions to streamline repetitive tasks and improve efficiency.
  • Keep the UI clean and organized by closing unnecessary tabs and windows.
